A Professional Portfolio using html and css

Tags Used 
 

1) Ul/Li tags used for navbar

2) Icon i tag used for github,linkedin,facebook,email,contact icons

3) img tag used for images 

4) main tag is used to display to main content

5) Footer tag is used for contact me information

6) Absolute properties are used for placement of the images 

7) Transition and transform are used to give a simple effect to the image ,buttons,navbar

8) Different hover styles are used to make the website fun 

9) Tel and mailto are used to redirect user to contact Me

10) The website is made responsive for all users

11)Figcaption included to include description of the image
